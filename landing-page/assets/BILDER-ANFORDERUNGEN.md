# Benötigte Bilder für die Landing Page

Um die Landing Page vollständig zu machen, benötigen Sie folgende Bilder:

## 📸 Erforderliche Bilder

### 1. Dashboard Screenshot
- **Datei**: `/images/dashboard-mockup.png`
- **Größe**: 800x600px (mindestens)
- **Format**: PNG oder JPG
- **Inhalt**: Screenshot des Translation Office Dashboards
- **Verwendung**: Hero Section (rechte Spalte)

### 2. Vorteile Illustration
- **Datei**: `/images/benefits-illustration.png`
- **Größe**: 600x500px (mindestens)
- **Format**: PNG oder JPG
- **Inhalt**: Illustration oder Screenshot für den Vorteile-Bereich
- **Verwendung**: Benefits Section (linke Spalte)

### 3. Open Graph Bild
- **Datei**: `/images/og-image.jpg`
- **Größe**: **Exakt 1200x630px** (Facebook/LinkedIn Standard)
- **Format**: JPG
- **Inhalt**: Logo + Text "Translation Office - TMS für Übersetzungsbüros"
- **Verwendung**: Social Media Vorschau (Facebook, LinkedIn, Twitter)

### 4. Favicon
- **Datei**: `/assets/favicon.svg`
- **Größe**: 32x32px (oder skalierbar als SVG)
- **Format**: SVG (bevorzugt) oder PNG
- **Inhalt**: Logo-Icon der Translation Office Marke
- **Verwendung**: Browser-Tab-Icon

### 5. Testimonial Profilbilder (Optional)
- **Dateien**: `/images/testimonial-1.jpg`, `/images/testimonial-2.jpg`, `/images/testimonial-3.jpg`
- **Größe**: 100x100px
- **Format**: JPG
- **Inhalt**: Profilbilder der zitierten Kunden (oder Platzhalter)
- **Verwendung**: Testimonials Section

## 🎨 Design-Richtlinien

### Farben
- **Primärfarbe**: #1B4D4F (Teal)
- **Sekundärfarbe**: #2A6A6D (Helles Teal)
- **Akzentfarbe**: #FFD700 (Gold/Gelb)

### Stil
- Modern und professionell
- Business-orientiert
- Klare, übersichtliche Screenshots
- Keine ablenkenden Elemente
- Hohe Qualität (mindestens 72 DPI für Web)

## 🛠️ Tools zum Erstellen

### Screenshots erstellen
- **Windows**: Snipping Tool / Windows + Shift + S
- **macOS**: Command + Shift + 4
- **Browser**: Chrome DevTools (F12) → Device Toolbar für responsive Screenshots

### Bildbearbeitung
- **Online**: Canva.com, Figma.com, Photopea.com
- **Desktop**: GIMP (kostenlos), Photoshop, Affinity Photo
- **Komprimierung**: TinyPNG.com, Squoosh.app

### OG-Image Generator
- https://og-image.vercel.app/
- https://www.bannerbear.com/tools/open-graph-image-generator/

### Favicon Generator
- https://realfavicongenerator.net/
- https://favicon.io/

## ✅ Checkliste

Sobald Sie die Bilder haben:

- [ ] Dashboard-Screenshot erstellt und optimiert
- [ ] Benefits-Illustration erstellt
- [ ] Open Graph Bild (1200x630px) erstellt
- [ ] Favicon (SVG oder PNG) erstellt
- [ ] Alle Bilder komprimiert (TinyPNG)
- [ ] Bilder in die richtigen Ordner kopiert
- [ ] Alt-Tags in HTML überprüft
- [ ] Bilder in verschiedenen Browsern getestet
- [ ] Mobile-Ansicht getestet

## 📝 Hinweise

- **Dateigrößen**: Halten Sie Bilder unter 200KB für schnelle Ladezeiten
- **Formate**: WebP für moderne Browser, JPG/PNG als Fallback
- **Responsive**: Testen Sie die Bilder auf verschiedenen Bildschirmgrößen
- **Copyright**: Verwenden Sie nur Bilder, für die Sie die Rechte haben

## 🚀 Schnellstart (Platzhalter)

Die Landing Page verwendet derzeit Platzhalter-Bilder von Placehold.co.
Diese werden automatisch angezeigt, bis Sie echte Bilder hochladen.

```html
<!-- Aktueller Platzhalter -->
<img src="..." onerror="this.src='https://placehold.co/800x600/1B4D4F/white?text=Dashboard+Screenshot'">
```

Sobald Sie echte Bilder haben, ersetzen Sie einfach die Dateien in den entsprechenden Ordnern!
